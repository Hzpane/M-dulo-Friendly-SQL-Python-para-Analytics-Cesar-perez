# Análisis de pedidos Delivery con Python y SQL

## Descripción del proyecto

Este proyecto desarrolla un análisis exploratorio de datos utilizando Python, pandas, Plotly y DuckDB sobre un dataset de pedidos delivery obtenido desde Kaggle.

El objetivo es identificar patrones relacionados con demanda, tiempos de entrega, métodos de pago y variables operativas del servicio.

## Dataset

Fuente:
Kaggle

Dataset:
Talabat Enhanced Orders Dataset

## Objetivo

Realizar un análisis exploratorio mediante:

- Limpieza y revisión de calidad de datos.
- Estadística descriptiva.
- Visualizaciones interactivas.
- Consultas SQL utilizando DuckDB.

## Tecnologías utilizadas

- Python
- Pandas
- NumPy
- Plotly
- DuckDB
- Google Colab

## Etapas del análisis

1. Carga del dataset.
2. Exploración inicial.
3. Análisis de valores nulos y duplicados.
4. Estadística descriptiva.
5. Visualización de datos.
6. Consultas SQL.
7. Conclusiones.
